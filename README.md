## 6. 회고 및 느낀점
>Java를 배우고 처음으로 만든 프로그램이라 완성하는데 힘들었고 이해가 안되는 부분이 많았습니다. 하지만 계속 보다 보니 약간씩 이해되기 시작하고 코드가 실행되는 방식이 조금씩 보이기 시작했다. 그래도 눈으로 보이는 프로그램을 만드니 재밌고 계속 연습하여 지금 프로그램 보다 훨씬 더 좋은 프로그램을 만들도록 노력하겠다.
